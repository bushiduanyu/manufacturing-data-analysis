# Manufacturing Data Analysis

This is my first Python-based engineering analytics project. The goal is to practice moving from spreadsheet-based analysis to reproducible data analysis using Python, with a focus on manufacturing and production-related data.

## Project Goal

The goal of this project is to learn how to use Python to read, clean, analyze, and visualize engineering or manufacturing datasets.

## Tools

- Python
- pandas
- NumPy
- matplotlib
- Jupyter Notebook
- GitHub

- Created the first GitHub repository
- Preparing to find and analyze a manufacturing-related dataset
